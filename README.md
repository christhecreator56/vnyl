# VNYL | Digital & Live Agency

A modern, minimalist website for VNYL - a digital and live agency specializing in brand experiences.

## Features

- **Custom Calendar**: Interactive date picker for scheduling meetings
- **Service Showcase**: Display of digital strategy, brand experience, and live events services
- **Philosophy Section**: Brand philosophy highlighting the balance between digital and analogue
- **Booking System**: Integrated meeting scheduler with time slot selection
- **Responsive Design**: Optimized for all device sizes
- **Custom Cursor**: Interactive cursor with hover effects

## Color Scheme

- **Primary Background**: Pale Yellow (#FEF9E7)
- **Accent Color**: Black (#0a0a0a)
- **Text**: Black on pale yellow, Pale yellow on black

## Technologies

- HTML5
- Tailwind CSS
- Lucide Icons
- Vanilla JavaScript

## Setup

Simply open `index.html` in a web browser. No build process required.
